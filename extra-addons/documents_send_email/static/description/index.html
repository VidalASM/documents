Send Document | Document Send by Email

This module offers easy-to-use capability for sending documents from the documents module. Additionally, if you navigate to the partners' documents and send documents as email, this module will record the message.

Installation

- Copy documents_send_email module to addons folder

- Install the module normally like other modules

1. Send Documents from Documents module

Documents -> Select Documents -> Send Mail
